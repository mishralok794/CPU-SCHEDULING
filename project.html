About the Project

  • The project demonstrates the working and visualization of various CPU Scheduling Algorithms used in Operating Systems.
  • Understanding scheduling algorithms is essential for Computer Science students to master process management and performance optimization concepts.
  • Scheduling algorithms determine the order in which processes are executed by the CPU — directly impacting system efficiency, response time, and throughput.
  • The project includes algorithms like FCFS, SJF, SRTF, Priority, Round Robin, LRTF, and HRRN.
  • Frontend technologies used: HTML for structure, CSS for design, and JavaScript for interactive functionality.
  • Developed collaboratively by Alka Jha and Alok Mishra under academic guidance.
